# ddi

## Data and plotting scripts

This repository contains numerical data and Python plotting scripts organized by figure folders.

## Contents

- `fig1/` – Data and plotting scripts for Fig. 1  
- `fig2_fig3/` – Data and plotting scripts for Figs. 2–3  
- `fig4/` – Data and plotting scripts for Fig. 4  
- `fig5/` – Data and plotting scripts for Fig. 5  
- `fig6/` – Data and plotting scripts for Fig. 6  
- `fig7/` – Data and plotting scripts for Fig. 7  

Each folder includes the raw data files and the corresponding Python scripts used to reproduce the plots.

## Requirements

- Python ≥ 3.9
- NumPy
- Matplotlib
- Pandas

(Optional) Jupyter Notebook for interactive runs.
